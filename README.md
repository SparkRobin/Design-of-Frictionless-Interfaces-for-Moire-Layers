# Mathematica notebooks for frictionless moire interfaces

Readable Mathematica notebooks for the symbolic calculations in:

Zichong Zhang and Shuze Zhu, *Design of Frictionless Interfaces for Moire Layers*, DOI: [10.1103/zqw2-sgfq](https://doi.org/10.1103/zqw2-sgfq).

## Files

- `01_density_function.nb`: translated moire density, SI Eq. (S1)-Eq. (S2).
- `02_parallelogram_energy.nb`: 60-degree parallelogram integral and integer frictionless limits.
- `03_magic_angles.nb`: second-kind magic-angle condition.
- `04_strain_engineering.nb`: strain-engineering formulas, SI Eq. (S21)-Eq. (S24).
- `05_hexagon_triangle.nb`: regular-hexagon and equilateral-triangle checks.
- `06_main_text_eq2.nb`: main-text Eq. (2) for a general parallelogram flake.
